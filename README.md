Esse projeto foi feito para analisar o desempenho de 4 lojas da Alura Store. O objetivo é ajudar o Sr. João a decidir qual loja vender com base em dados reais de vendas.

## Objetivo

Avaliar as lojas usando as seguintes métricas:
- Faturamento total
- Frete médio
- Avaliação dos clientes
- Produtos mais e menos vendidos
- Quantidade de vendas por categoria

## Ferramentas usadas

- Python
- Google Colab
- pandas
- matplotlib

## Conclusão

Depois de analisar os dados, identifiquei qual loja teve o pior desempenho em comparação com as outras, e isso foi usado para recomendar ao Sr. João qual ele pode vender.
